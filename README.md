# 🎯 Prospection Next.js

## Objectif
Trouver des prospects (artisans + fast-food) sans site ou avec site obsolète (WordPress mal optimisé) pour proposer création/refonte en Next.js.

## Argumentaire clé
- ⚡ **Vitesse** : Next.js = sites ultra-rapides (Core Web Vitals excellents)
- 🔍 **SEO naturel** : SSR/SSG = meilleur référencement Google
- 💰 **Conversion** : +30% de conversions sur mobile grâce à la vitesse
- 🔒 **Sécurité** : moins de failles qu'un WordPress non maintenu

## Structure du vault
- [[Artisans]] → Fiches par métier (plombier, électricien, menuisier...)
- [[Fast-Food]] → Fiches par type (burger, kebab, pizza, snack...)
- [[Templates]] → Email de prospection, scripts d'appel, audit rapide
- [[Suivi]] → Tableau de suivi des contacts envoyés

## Méthodologie de recherche
1. Cibler une zone géographique (ville ou département)
2. Chercher sur Google Maps / Pages Jaunes / Yelp
3. Vérifier si site web existe (sinon = prospect chaud)
4. Si site existe : analyser rapidement (PageSpeed, WordPress ?)
5. Noter : nom, adresse, téléphone, email, site, défauts identifiés
6. Envoyer email personnalisé avec audit rapide
